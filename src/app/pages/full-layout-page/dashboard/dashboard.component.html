Instructions for ATSL 2020 Trial Test

  • ATSL 2020 Trial test will start at 2:00 PM on 24th January 2020
  • Students can login using their assigned Username & Password
  • All questions are of multiple choice type
  • The questions will be displayed one by one
  • The student has to select one of the choices and click on “Next”
  • After clicking “Next” the student cannot go back to change the answer
  • After completing 90 questions, the student has to click on “Complete Test”
  • Each student can take the test only once
  • If the student closes, or clicks refresh or if the internet connection is lost during the exam, the student will have to Restart the exam. It is not possible to continue the previously started test.
